# Open Decisions

## Decision 1: Single HTML vs. Framework
**Chosen:** Single HTML file with vanilla JS
- Simpler deployment
- Easier to understand/modify
- No build step required

## Decision 2: PDF Library
**Chosen:** jsPDF via CDN
- Well-supported
- No build step

## Decision 3: localStorage Only
**Chosen:** No backend
- MVP only
- Tradesmen have their own devices
- Can export/import JSON backup later
