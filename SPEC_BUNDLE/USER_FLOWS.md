# Solo Service Invoice - User Flows

## Flow 1: First-Time Setup
1. User opens app
2. Prompted to enter business name
3. Business info saved
4. Dashboard shown (empty state)

## Flow 2: Create Invoice (New Customer)
1. Click "New Invoice" button
2. Enter customer name
3. Enter customer address
4. Enter service description
5. Add line items (tap "+ Add Item")
   - Enter service name
   - Enter quantity
   - Enter rate
   - Total auto-calculates
6. Tap "Create Invoice"
7. PDF generated automatically
8. Download prompt shown
9. Redirect to dashboard

## Flow 3: Create Invoice (Saved Customer)
1. Click "New Invoice"
2. Tap "Select Customer"
3. Choose from saved list
4. Customer info auto-fills
5. Continue from step 4 above

## Flow 4: Save Customer
1. During invoice creation
2. Check "Save customer for later"
3. Customer saved
4. Can select on next invoice

## Flow 5: View Dashboard
1. View total revenue
2. View invoice count
3. Tap invoice to view details

## Flow 6: Manage Customers
1. Go to Customers tab
2. View all saved customers
3. Tap to edit
4. Or tap delete icon
